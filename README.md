## 프로젝트 기간 : 2023.07.12 ~ 2023.08.31 (아이데이션 기간 포함) 
## 프로젝트 소개 : 교통약자를 위한 저상버스 탑승 예약 서비스
- 특정 정류소에서 저상버스 탑승예약을 할 수 있습니다.
- 탑승 예약시 해당 버스의 버스기사에게 알람이 갑니다.
- 승하차 완료시 후기 작성이 가능합니다.
